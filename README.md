# UI Button Replication Practice

A beginner-friendly HTML and CSS project focused on replicating the exact visual styles of core buttons from well-known platforms.

## 🎯 Purpose of the Project
The goal of this project was purely foundational. I wanted to practice **pixel-perfect styling** by trying to match the exact background colors, padding, border-radius, and font styles used by major companies. 

Instead of building a full webpage, I focused strictly on mastering the details of individual interactive elements.

## 📁 Replicated Buttons:
* **Uber:** "Request now" (Black button layout)
* **Amazon:** "Add to Cart" (Yellow button layout)
* **LinkedIn:** "Apply on company website" (Blue button layout)
* *(Plus other various CTA buttons)*

## 🛠️ Tech Stack
* **HTML5:** For the basic button markup.
* **CSS3:** For matching hex colors, margins, and padding.

As I learn JavaScript, I plan to come back to this repository and add functional click alerts or hover transitions to these buttons!
